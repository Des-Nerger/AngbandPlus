How to build all libraries required to run the WIN client:

- Download and unzip ZLIB 1.2.7 in the same root directory as your \PWMAngband folder:
	http://zlib.net/
- Download and unzip LIBPNG 1.5.12 in the same root directory as your \PWMAngband folder:
	http://sourceforge.net/projects/libpng/files/
- Edit makefile.bc32 in the lpng1511\scripts directory:
	set ZLIB_DIR=..\zlib-1.2.7
- Run \PWMAngband\src\WIN.bat
